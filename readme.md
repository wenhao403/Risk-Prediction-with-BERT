# 獨居老人風險預測

## 專案介紹
專案目標為利用獨居老人的身心健康資料建立風險警示系統，以達到事先的預防，並透過人力的介入與關懷來降低獨居老人孤老死亡的可能。\
(由於專案涉及隱私，不便透露過多，**故內容主要紀錄BERT模型的前處理與使用方式以及相關資源分享**)
